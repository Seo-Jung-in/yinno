Parylene Types

용도와 환경에 따라 최적의 파릴렌 유형을 선택하세요

CVD Coating Advantages Parylene Types Properties
PARYLENE TYPES

파릴렌 코팅 유형 비교

파릴렌은 분자 구조에 따라 N, C, D, HT, AF-4 등 여러 유형으로 나뉩니다.
각 유형의 특성을 확인하고 용도에 맞는 최적 유형을 선택하세요.

Type N

Parylene N

고주파 전자회로·MEMS 소자에 최적화된 기본형

치환기 없는 순수 파릴렌. 높은 유전강도(7,000 V/mil 이상)와 매우 낮은 WVTR이 특징입니다. 정밀 고주파 회로에서 신호 손실을 최소화합니다.

  • 유전강도 7,000 V/mil 이상
  • 온도범위 -200°C ~ 80°C
  • 낮은 WVTR (0.59 g·mm/m²/day)
  • 고주파·MEMS 응용 최적
Type C

Parylene C

PCB·의료기기·전자부품 제조사가 가장 많이 선택

Monochloro 치환 구조. 우수한 수분·가스 차단성과 화학적 내성을 갖추어 전자부품 및 의료기기 분야 범용으로 사용됩니다. USP Class VI, ISO 10993 인증.

  • USP Class VI / ISO 10993 인증
  • 온도범위 -65°C ~ 130°C
  • WVTR 0.08 g·mm/m²/day
  • 의료 임플란트·PCB 보호
Type D

Parylene D

자동차 엔진룸·고온 산업 장비 제조사에 적합

Dichloro 치환 구조로 파릴렌 C 대비 향상된 내열성을 발휘합니다. 100°C 이상의 고온·화학 환경에서도 코팅 성능이 유지됩니다.

  • 온도범위 -100°C ~ 150°C
  • 높은 기계적 강도
  • WVTR 0.10 g·mm/m²/day
  • 고온·화학 환경 특화
Type HT

Parylene HT

항공우주·방산·LED 고온 응용에 필수

불소 치환 구조로 최고 수준 열 안정성을 발휘합니다. 야외·우주 환경의 강한 UV와 극한 온도 변화에서도 안정적입니다.

  • 온도범위 -200°C ~ 350°C
  • 최고 수준 UV 안정성
  • WVTR 0.02 g·mm/m²/day
  • 항공우주·방산 인증 환경
Type AF-4

Parylene AF-4

우주·화학 극한 환경 및 초저 유전율이 필요한 분야

Tetrafluoro 치환 구조로 최저 유전율(2.0)을 달성합니다. 우주 환경과 극한 화학 환경에서 최고 수준의 안정성과 UV 내성을 제공합니다.

  • 최저 유전율 2.0 (1MHz)
  • 온도범위 -200°C ~ 400°C
  • WVTR 0.01 g·mm/m²/day
  • 최고 수준 UV·방사선 안정성
