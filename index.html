파릴렌 코팅 기술을
선도하는 기업

주식회사 와인(Y-INNO)은 경상남도 창원시 SK테크노파크에 위치한 파릴렌(Parylene) CVD 코팅 전문 기업입니다.

화학기상증착(CVD) 방법을 이용한 파릴렌 고분자 코팅 기술을 바탕으로, 전자·의료·디스플레이·자동차·항공우주·산업 분야 고객에게 최고 품질의 보호 코팅 솔루션을 제공합니다.

PARYLENE TECHNOLOGY

파릴렌 코팅의 핵심 특성

파릴렌은 진공 챔버 내 화학기상증착으로 형성되는 고분자 박막 코팅으로,
극한 환경에서도 탁월한 보호 성능을 발휘합니다.

우수한 보호성

수분·산소 차단성과 내식·방청 성능을 통해 외부 환경으로부터 부품을 안정적으로 보호합니다.

수분 차단산소 차단방청

균일 코팅성

핀홀 없는 균일한 컨포멀막을 형성하며, 복잡한 3D 구조나 미세 부품에도 균일하게 코팅됩니다.

컨포멀 코팅핀홀 無3D 구조

전기 절연성

높은 절연성과 저유전율로 고전압 환경에서 고신뢰성 전자 부품을 안정적으로 보호합니다.

고절연저유전율고신뢰성

생체 적합성

USP Class VI 인증 생체적합성으로 의료 임플란트 및 피부 접촉 기기에 최적화되어 있습니다.

PARYLENE TYPES

파릴렌 코팅 유형

용도와 환경에 따라 최적의 파릴렌 유형을 선택할 수 있습니다.

Type C

파릴렌 C

가장 널리 사용되는 유형으로, 우수한 수분·가스 차단성과 화학적 내성을 제공합니다. 전자부품 보호에 최적화되어 있습니다.

  • 우수한 수분 차단성
  • 높은 화학적 내성
  • 전자부품 최적화
Type D

파릴렌 D

파릴렌 C 대비 높은 내열성을 가지며, 고온 환경에서도 안정적인 성능을 발휘합니다.

  • 우수한 내열성
  • 고온 환경 적합
  • 기계적 강도
Type F

파릴렌 F

불소 함유 구조로 뛰어난 내화학성과 낮은 유전율을 제공하며, 고주파 전자 응용에 적합합니다.

  • 초저 유전율
  • 우수한 내화학성
  • 고주파 응용
Type HT

파릴렌 HT

최고 수준의 내열성과 UV 안정성을 보유하며, 항공우주·방산 분야의 극한 환경에 최적입니다.

  • 최고 수준 내열성
  • UV 안정성
  • 항공·방산 적합
